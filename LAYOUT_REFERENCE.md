## Atajos Rápidos

### Navegación (mantener LOWER)
| Tecla | Función |
|-------|----------|
| H | ← Izquierda |
| J | ↓ Abajo |
| K | ↑ Arriba |
| L | → Derecha |
| HOME | Inicio de línea |
| END | Fin de línea |
| PGUP | Página arriba |
| PGDN | Página abajo |

### Símbolos (mantener RAISE)
| Tecla | Símbolo |
|-------|---------|
| - | Guion |
| = | Igual |
| [ | Corchete izquierdo |
| ] | Corchete derecho |
| \ | Barra invertida |
| ' | Comilla simple |
| " | Comilla doble |
| ; | Punto y coma |
| | | Barra vertical |

### Sistema (LOWER + RAISE)
| Tecla | Función |
|-------|---------|
| PRTSC | Captura pantalla |
| RGB_TOG | Encender/apagar RGB |
| RGB_HUD/HUI | Cambiar tono |
| RGB_BRD/BRI | Cambiar brillo |
| BT_SEL 0-4 | Cambiar perfil Bluetooth |
| BT_CLR | Limpiar perfiles |
| BOOT | Modo bootloader |
| RESET | Reiniciar |

## Flashing

1. Descarga firmware desde GitHub Actions
2. Descomprime `firmware.zip`
3. Flashea cada lado:
   - `settings_reset.uf2` → ambos lados (primero)
   - `Lily58_left_oled.uf2` → lado izquierdo
   - `Lily58_right_oled.uf2` → lado derecho

## ZMK Studio

1. Conecta el lado izquierdo por USB
2. Abre https://zmk.studio
3. El teclado debería detectarse automáticamente

## Notas

- OLED muestra información del teclado
- RGB se controla desde ADJUST layer
- Para español, usar tecla `;` para Ñ (código SEMI)
- El fix `CONFIG_ZMK_SLEEP=n` previene que el OLED se apague
